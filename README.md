# rapp-eternity — the rappid identity & ownership standard

`rapp-eternity/1.0`: a rappid is a **permanent, PKI-free content address** (full SHA-256 of canonical
content), owned by default through **GitHub collaboration + file possession**, where a keypair is an
**optional** sovereignty upgrade that **no component may ever require**. Unifies the divergent rappid
forms across the estate; read-all-legacy, emit-only-canonical. See **[SPEC.md](SPEC.md)**.
